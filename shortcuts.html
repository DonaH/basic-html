Shortcuts I've Learned So Far

Symbols

  • The command symbol looks like a flower.
  • The control symbol looks like a hat or carrot.
  • The shift symbol looks like an arrow pointing up.

Shortcuts

  • Command / (that's a forward slash) comments out something in an html page.
  • Command "V" pastes something that has been copied or cut somewhere on a page.
  • In Terminal, Control Command "L" clears your screen, but does not delete all of the previous commands you have typed!
  • Command Z means undo. If you make a mistake, you can undo it one or more times.
  • In Terminal, if you begin typing a word, usually about three letters or characters in, if you stop and hit TAB, it'll auto-fill for you.
